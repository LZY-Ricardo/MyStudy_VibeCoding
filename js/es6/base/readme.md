# let, const vs var
1. let声明的变量不存在声明提升, var 存在
2. var 在全局声明的变量, 会成为 window 的属性, let 不会
3. let 不允许重复声明同一个变量, var 可以
4. let + {} 会形成块级作用域, var 不会
5. const 声明的变量必须要赋值, 不能不赋值, 也不能改变
