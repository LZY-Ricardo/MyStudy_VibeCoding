- npm i -g expo-cli

## rn
来自 FaceBook 的移动端开发框架
- react SPA 框架、 SSR web 网页应用 是前端 80% 工作主流
- IOS/Android 开发的移动端应用
  应用市场
  - IOS/Android 开发, 一个应用要搞两波人
    - Java\kotlin
    - Objective-c/Swift
    开发成本极高
  - rn 跨平台开发, 一套代码 可以同时在 IOS/Android 上运行
    性能很好
    js bridge react 写的代码 调用手机上的设备
    VDOM 
- expo 是让rn开发更丝滑的工具
  - expo go 扫码安装应用
  - 手机上查看效果 react 开发

  ### 思路切换
